# CMS_Frontend

## Overview
A Vue 3 frontend application built with Vite for a CMS system


## Tech Stack
- Vue 3
- Vite
- TypeScript
- Axios

## Backend

The Backend API for this project is located at a separate repo

[C# Backend API](https://github.com/FraChr/CMS-Backend/tree/main)

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
